# md-to-html

CLI tool to convert Markdown (footnote supported) to HTML.

## Installation

```
npm install -g @kubosho/md-to-html
```

## Usage

Output HTML to stdout:

```
md-to-html foo.md
```

Output HTML to file:

```
md-to-html foo.md --to path/to/foo.html
```

## Options

### `--to <filepath>`

Output HTML file in specified path.
